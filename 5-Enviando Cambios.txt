para enviar un commit y registrarlo en el repositorio

    git commit

   comando :wq   luego enter


 o la opcion de 

     git commit -m "mensaje del commit"
  
    
Nos aparecera un espacio con unas lines de muestra para
poder escribir una descripcion del commit 

Recomendacion de comentarios 

1-Por que has hecho este cambio

2-que cambios has cambiado , descripcion rapido , nombres de archivos etc

3-Que consecuencias esperas que tenga tu cambio ?
  ejemplo:
   ya se podra hacer xxxxx 
   o esto afecta como se comunincan los modulos x o y
