Para crear una rama apartir de la rama y
 commit actual 

    git branch nombre_rama


para ver las ramas existentes y en cual de 
ellas se esta actualmente

    git branch


para ver las ramas de un repositorio remoto , porque normalmente solo se ve la master

    git branch -a


para cambiar de rama

    git checkout nombre_rama



para crear una nueva rama y moverse inmediatamete
a ella

    git checkout -b nombre_rama        



para borrar una rama 

    git branch -d nombre_rama


para cambiar el nombre de una rama existente 

    git branch -m [oldName] [newName]
